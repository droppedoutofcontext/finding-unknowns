Скилл для [Claude Code](https://claude.com/claude-code), который превращает расплывчатые промпты в
точные спеки. Вместо того чтобы гадать и сразу строить, Claude недорого тратит пару минут на маленький
**интерактивный HTML-артефакт**, который помогает тебе *узнать* то, что ты не мог сформулировать, —
и **сам пишет твоё следующее сообщение.**

## Техники

Люди плохо *формулируют* свои стандарты, но отлично их *узнают* — покажи четыре отрисованных варианта,
и человек ткнёт мгновенно; спроси «какой стиль?» — пожмёт плечами. Claude выбирает подходящую технику и
строит под неё артефакт. Блюпринты — в [`SKILL.md`](skills/finding-unknowns/SKILL.md).

| Фаза | Техника | Что вытаскивает из твоей головы |
|------|---------|---------------------------------|
| До | Blindspot pass | unknown unknowns → вставки в промпт |
| До | Teach me my unknowns | словарь для вкуса, который не назвать |
| До | Four design directions | ткни в отрисованный вид; steal/skip по элементам |
| До | Mock before you wire | размещение и взаимодействие — до кода |
| До | Brainstorm on an effort axis | что заходит: от «за полдня» до «на квартал» |
| До | The interview | решения по архитектурной цене ошибки |
| До | Point at a reference | проверка: референс понят до портирования |
| До | The tweakable plan | согласование плана по вероятности правок |
| Во время | Implementation notes | расхождения плана с реальностью — для попытки №2 |
| После | The buy-in doc | возражения ревьюера, отвеченные заранее |
| После | Quiz me before I merge | понимаешь ли *ты* свой дифф? |

## Установка

```
/plugin marketplace add droppedoutofcontext/finding-unknowns
/plugin install finding-unknowns@finding-unknowns
```

Либо скопируй папку скилла напрямую — `cp -r skills/finding-unknowns ~/.claude/skills/`. Это скилл
стандартного формата `SKILL.md`, поэтому его подхватят и другие агенты, понимающие формат Agent Skills.

## Использование

Обычно вызывать вручную не нужно — срабатывает на расплывчатых, эстетических или рискованных запросах
(«сделай поприятнее», «добавь шеринг, детали на твоё усмотрение», «я готов к мержу?»). Чтобы позвать
напрямую, назови технику или скилл:

> *«проинтервьюируй меня об этой фиче»* · *«сделай blindspot pass по этому модулю»* ·
> *«прогони меня квизом перед мержем»* · `/finding-unknowns`

## А это работает?

Четыре реалистичные задачи, по четыре бинарные проверки в каждой — **16/16** со скиллом против **3/16**
у базы. База часто *находит* те же проблемы; преимущество скилла в петле, а не в проницательности — он
превращает находки в твои решения и твой следующий промпт вместо стены текста. Выборка маленькая;
перезапустить можно через [`evals/`](skills/finding-unknowns/evals/evals.json). Протестировано на
Fable 5 и Opus 4.8.
